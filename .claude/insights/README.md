# Insights Collection

这里记录aiNagisa开发过程中的启发、想法和概念。

## 文件组织

- `shadow-learning-concept.md` - Shadow Learning模式设计
- 更多insights将以类似格式记录...

## 命名规范

- 使用描述性的kebab-case命名
- 包含日期戳：`YYYY-MM-DD-concept-name.md`
- 或主题分类：`agentic-thinking-patterns.md`

## 这种方式的优势

1. **版本控制**: 想法的演进可追溯
2. **就近原则**: 代码旁边记录想法
3. **协作友好**: 团队都能看到和贡献
4. **搜索便利**: grep/rg直接搜索想法
5. **无额外工具**: 不依赖外部note-taking工具

## 与传统笔记工具的区别

| 传统工具 | .claude/insights |
|---------|------------------|
| 独立存储 | 与代码共存 |
| 工具锁定 | 纯文本 |
| 同步麻烦 | git自动同步 |
| 难以搜索 | 代码级搜索 |
| 个人私有 | 团队共享 |

这就是未来的协作方式 - **思维与代码同步演进**。
