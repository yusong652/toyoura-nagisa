## Key Implementation Details

### Anthropic Best Practice Format
```
In this environment you have access to a set of tools...

[Formatting Instructions]
String and scalar parameters should be specified as is...

Here are the functions available in JSONSchema format:
<functions>
<function>{"name": "tool_name", "description": "...", "parameters": {...}}</function>
...
</functions>

[User System Prompt]
[Additional Context]
```

- **Target**: Centralized in `prompt_builder.py`, tools embedded in system prompt
